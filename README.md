# Weather App

A responsive weather web app that displays real-time weather data and forecasts using the Open-Meteo API.

## Features
- Current weather (temperature, feels like, conditions, wind, UV index)
- Hourly forecast with accurate local times
- 7-day forecast with highs and lows
- Geolocation support (defaults to Kent, Ohio if unavailable)
- Dynamic backgrounds based on weather conditions
- Toggle between Fahrenheit and Celsius
- Clean, responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Open-Meteo API
- Geolocation API

## How It Works
The app uses the browser’s Geolocation API to get the user’s location and fetches weather data from Open-Meteo. It dynamically updates the UI with current, hourly, and daily forecast data.

## Purpose
This project was created for a class assignment to demonstrate API integration, dynamic UI updates, and responsive design.
